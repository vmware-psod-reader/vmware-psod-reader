PSOD Debugger

MCE errors are caused by hardware failures and cause ESXi servers to PSOD. The MCi_STATUS register is useful in debugging the PSOD.

The MCi_STATUS register consists of information about the machine check error. The register is displayed next to the “S:” as shown in the example below.

MC: PCPU18 B:13 S:0xfe20004000011166 M:0x7246040086 A:0x38c989b100 5

Reference: VMware KB - Decoding Machine Check Exception (MCE) output after a purple screen error (1005184)

How to Debug:

Simply input the last 4 characters of the MCi_STATUS code below and click Debug.
